# easy-server
easy-server

# 使用
安装
```js
npm install easy-servers -g
```

使用
```js
easy server [指定文件夹]
```

执行 `easy -h`
```js
  Usage: easy [options] [command]

  Options:

    -h, --help  output usage information

  Commands:

```
